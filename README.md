Мультитаск-пайплайн машинного обучения для автоматического извлечения характеристик археологических находок по изображению и генерации структурированного текстового описания (`auto_description`). 

Проект разработан в рамках сотрудничества с лабораторией «Искусство и искусственный интеллект» ЕУСПб, ИИМК РАН и НовГУ. 
**Результат защиты решения — 98 баллов из 100.**


## Архитектура и Инженерные решения

* **Multi-Task Learning:** Реализована единая предобученная CV-модель (Backbone) с независимыми линейными «головами» (Linear Heads) для одновременного предсказания 4 полей: *Тип предмета*, *Часть/Зона*, *Целостность*, *Материал*.
* **Data-Centric EDA:** Выявлено и учтено 3 визуальных режима съемки (Multi-view карточки, Close-up клейм, одиночные фрагменты). Внедрен алгоритм **Resize Longest Side + Pad** для сохранения пропорций и защиты от утечек (Data Leakage) через фон и линейки.
* **Борьба с дисбалансом:** Применен динамический расчет `class weights` внутри функции потерь `CrossEntropyLoss` для выравнивания редких классов из длинного хвоста распределения.
* **Group-Aware Валидация:** Сплит данных (`train_inner` / `val_inner` / `test_open`) настроен через `GroupKFold` по ключу `group_key`, что полностью исключило попадание разных ракурсов одного предмета в разные выборки.
* **Логика уверенности:** Сборка текста `auto_description` управляется порогами уверенности модели (Confidence Thresholds). Если скор предсказания ниже порога, поле автоматически опускается во избежание галлюцинаций.
